## Setup

```
$ git clone https://github.com/hariNEzuMI928/priority-score
$ cd priority-score
$ npm i
$ npm run build
```

watch
```
$ npm run serve
```

## Browser settings

1. `npm run build`を実行する
2. chrome://extensions/ にアクセス
3. 右上の「デベロッパーモード」をON
4. 左上の「パッケージ化されていない拡張機能を読み込む」を押下
5. 当リポジトリのdistディレクトリを指定
6. 読み込んだ拡張機能を有効にする
